Hangman Game  ||Word Guessing Game
==================

Description:
-------------
The Word Guessing Game is a fun and interactive word guessing game where players have to guess a randomly selected word from various categories. The game offers multiple categories such as programming, movies, people, countries, animals, fruits, colors, cars, and superheroes. Players need to guess the word by clicking on letter boxes corresponding to the letters in the word.

Features:
-----------
- Multiple categories with a wide range of words to guess.
- Real-time countdown timer for added challenge.
- Keeps track of correct and wrong attempts.
- Sound effects for win, loss, and incorrect guesses.
- Ability to save and display the player's score.

Instructions:
---------------
1. When the game loads, it will randomly select a word from one of the available categories.
2. Click on the letter boxes to guess the letters in the word.
3. If the clicked letter is present in the word, the corresponding letter boxes will be filled in.
4. If the clicked letter is not present in the word, the incorrect attempt counter will increase, and the hangman drawing will progress.
5. The game ends when either the player correctly guesses the entire word or the countdown timer reaches 0.
6. After the game ends, the player can choose to start a new game or exit.
